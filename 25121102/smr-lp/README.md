# SMR Landing Page

飲食店向け売上・予約管理システム「SMR」のランディングページ実装です。

## セットアップ

環境に Node.js がインストールされていることを確認してください。

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## 構成

- `src/components/sections/`: 各セクションのコンポーネント
- `src/components/ui/`: 共通UIコンポーネント (Button, SectionWrapper)
- `src/styles/`: グローバルスタイルと変数定義

## デザインの特徴

- **ダーク/ライトモード**: セクションごとに背景色を切り替え、リズムを作っています。
- **アクセントカラー**: シアンブルー (#06B6D4) を使用し、テック感を演出。
- **CTA**: オレンジ〜赤のグラデーションで視認性を最大化。
